D3 JS Portfolio

These are examples of D3 JS visualizations which I've completed. I am using D3 v5 for all visualizations.

Super Bowl Games visualized with a force related graph

This force related graph shows all Super Bowl Games. Each node can be dragged across the screen or pinned to screen. To pin, double click node. The size of the node represents the number of Super Bowl appearances by team. Data set included a 1 or 0 value for each game, this was used to style the link between teams.
